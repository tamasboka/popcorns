## API endpointok és adatbázis

### Lista és rövid leírás
1) GET api/movies
   - Filmek listája
2) GET api/movies/{id}
   - Egy film adatai ID alapján
3) POST api/movies
    - Új film hozzáadása
4) POST | PATCH api/movies/{id}
   - Egy film adatainak frissítése
5) DELETE api/movies/{movie} 
   - Film törlése ID alapján
